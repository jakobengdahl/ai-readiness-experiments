# AI Readiness Experiments

This repository contains experiments, prototypes, and practical notes related to AI-ready statistical dissemination. It is intended as a working space for testing how official statistics and related data flows can be exposed, structured, and consumed effectively through modern AI assistants and agent-based interfaces.

## Initial scope

- Testing how statistical APIs and dissemination formats can be made easier for AI assistants and agents to discover and use.
- Exploring practical patterns for exposing metadata, structure, and documentation around official statistics in AI-friendly ways.
- Prototyping workflows where AI systems support retrieval, interpretation, and communication of statistical outputs.
